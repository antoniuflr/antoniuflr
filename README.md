## What I've built

- **[Rail-Crack](https://github.com/antoniuflr/Rail-Crack)** — OpenGL game where you repair cracked rails before the train gives up on you. C++.
- **[Temu-Chicken-Invaders](https://github.com/antoniuflr/Temu-Chicken-Invaders)** — a knockoff of Chicken Invaders with a grid-based ship builder. C++.

- **[Router-in-C](https://github.com/antoniuflr/Router-in-C)** — router software supporting IPv4 routing, ARP and ICMP. C.
- **[Custom-Switch](https://github.com/antoniuflr/Custom-Switch)** — custom switch logic running on Mininet. Networks coursework.
- **[Memory-Allocator](https://github.com/antoniuflr/Memory-Allocator)** — custom malloc/calloc/realloc/free as a shared library. Uses brk/sbrk for small heap allocations and mmap/munmap for large ones. C.
- **[HTTP-Client-for-a-REST-API](https://github.com/antoniuflr/HTTP-Client-for-a-REST-API)** — command-line HTTP client in C++ that talks to a REST API simulating an online movie library.
- **[TCP-UDP-Server](https://github.com/antoniuflr/TCP-UDP-Server)** — UDP clients publish messages on topics, subscribed TCP clients receive them. C++.
- **[Parallel-Firewall](https://github.com/antoniuflr/Parallel-Firewall)** — multi-threaded packet firewall using POSIX threads. C.
- **[sat-solver](https://github.com/antoniuflr/sat-solver)** — fast CNF SAT solver written in C.
- **[primul-site](https://github.com/antoniuflr/primul-site)** — my first website. HTML.
